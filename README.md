# AI/ML Portfolio

A curated portfolio of applied AI, machine learning, deep learning, deployment, and GenAI projects originally developed in Google Colab and reorganized for GitHub review.

## Projects

| Project | Area | Tools | Status |
|---|---|---|---|
| [FoodHub Order Analysis](projects/foodhub-order-analysis/) | Python, EDA, business analysis | Python, pandas, seaborn | Complete |
| [AllLife Bank Personal Loan Prediction](projects/alllife-bank-personal-loan-prediction/) | Supervised ML, classification | Python, scikit-learn, decision trees | Complete |
| [EasyVisa Approval Prediction](projects/easyvisa-approval-prediction/) | Ensemble ML, model tuning | Python, scikit-learn, XGBoost | Complete |
| [ReneWind Turbine Failure Prediction](projects/renewind-turbine-failure-prediction/) | Neural networks, predictive maintenance | Python, TensorFlow/Keras | Complete |
| [ExtraaLearn Lead Conversion Deployment](projects/extraalearn-lead-conversion-deployment/) | ML deployment, lead scoring | Flask, Streamlit, Docker, scikit-learn | Complete |
| [Healthcare RAG Question Answering](projects/healthcare-rag-question-answering/) | NLP, GenAI, RAG | Python, LangChain, embeddings, LLM prompting | Complete |

## Repository Structure

```text
AIML_Portfolio/
  README.md
  projects/
    project-name/
      README.md
      notebooks/
      data/
      models/
      reports/
      assets/
  .gitignore
```

## How To Review

Open each project folder and start with its `README.md`. The README explains the business problem, dataset, approach, key outcomes, and where to find the notebook or rendered report.

## Dataset Usage

The datasets in this repository were provided as part of Great Learning project coursework and are included for educational and portfolio demonstration purposes based on permission confirmed by Great Learning.

## Notes

- The original Colab submission filenames were renamed into shorter portfolio-friendly names.
- Local runtime folders such as `.venv/`, `__pycache__/`, checkpoint folders, and zip archives are excluded.
- Datasets are included only when they are directly used by a portfolio project.
